# 智能体 v2 - 使用说明

## 快速开始

### 1. 配置密钥
打开 `隐私区/我的配置/密钥.json`，填入你的API Key和模型名称：
```json
{
  "密钥列表": {
    "LLM_API_KEY": "你的API Key",
    "LLM_MODEL": "gpt-4"
  }
}
```

### 2. 配置模型接口
打开 `公共区/配置/模型规则.json`，填入接口地址：
```json
{
  "接口地址": "https://api.example.com/v1/chat/completions"
}
```

### 3. 启动
- Windows: 双击 `启动.bat`
- Linux/Mac: `bash 启动.sh`

### 4. 使用
浏览器打开 http://localhost:8765

## 功能说明

| 功能 | 说明 |
|------|------|
| 💬 对话 | 与大模型对话 |
| 📁 文件管理 | 浏览文件、授权权限 |
| 📝 编辑器 | 打开编辑文件、语法高亮 |
| 🧠 记忆 | 查看事件记忆和用户画像 |
| 📋 日志 | 查看LLM原始请求响应 |
| ⚙️ 引擎 | 管理双引擎、合并代码 |
| 🔧 配置 | 编辑所有JSON配置 |

## 权限说明
- 智能体访问任何新文件夹都会弹窗询问
- 可选择：允许一次 / 永久允许 / 拒绝
- 同一路径询问3次后自动升级为永久授权
- 权限保存在 `公共区/配置/文件权限.json`
