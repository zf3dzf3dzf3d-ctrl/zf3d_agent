# 智能体 v2 - 使用说明

## 快速开始

### 1. 配置密钥
打开 `隐私区/我的配置/密钥.json`（从 `密钥_模板.json` 复制），填入你的API Key：

```json
{
  "密钥列表": {
    "DeepSeek(深度求索)": {
      "API密钥": "你的API Key",
      "模型名称": "deepseek-chat"
    },
    "TAVILY": {
      "API密钥": "你的Tavily API Key（可选，tavily.com免费注册）",
      "模型名称": ""
    }
  }
}
```

> 也可启动后在 **设置 → 模型** 页面直接配置：
> - 模型密钥：选择模型后填写，保存后加密存储
> - 工具密钥：Tavily等非LLM工具的API Key，界面内查看/保存

### 2. 启动
- **Windows**: 双击 `启动.bat`
- **Linux**: `bash 启动.sh` 或先 `bash install.sh` 一键安装
- **命令行**: `python3 公共区/内核/启动器.py`

### 3. 使用
浏览器打开 http://localhost:8765

## 权限说明
- 智能体访问任何新文件夹都会弹窗询问
- 可选择：允许一次 / 永久允许 / 拒绝
- 同一路径询问3次后自动升级为永久授权
- 权限保存在 `公共区/配置/文件权限.json`

## 文件管理操作
- **拖拽移动**：拖拽文件/文件夹到左侧目录树或中间画廊文件夹 = 移动
- **Shift+拖拽复制**：按住Shift拖拽 = 复制到目标文件夹
- **悬浮按钮**：鼠标悬停缩略图右上角出现 🗑️删除 / 📦移动到 / 📋复制到
- **框选批量**：空白区域拖拽框选，Ctrl+点击加减选，Ctrl+A全选，批量拖拽移动/复制
- **分页加载**：文件夹内容过多时自动分页（每页200项），底部点击加载更多

## Tavily搜索引擎
- **作用**：AI优化的网络搜索，返回干净正文+相关度评分+总结答案
- **获取**：https://tavily.com 免费注册，每月1000次搜索
- **配置**：在 设置 → 模型 → 工具密钥 中填入，或编辑 `密钥.json`
- **回退**：未配置时自动使用Bing搜索/urllib抓取，功能不受影响
