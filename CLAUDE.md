# ZF3D Agent v2 — 项目描述

## 根目录
`C:\Users\Administrator\Desktop\zf3d_Agent\新系统_v2`

## 项目简介
这是一个全JSON配置驱动的AI智能体框架。系统是手脚（调度、搬运、执行），大模型是大脑（思考、决策、优化），用户是主人（授权、审批、合并）。

## 技术栈
- Python 3.8+（纯标准库，零外部依赖）
- 前端：纯HTML/CSS/JavaScript（无框架）
- LLM接口：兼容OpenAI格式（通过urllib直连，不依赖SDK）
- 存储：全JSON文件（无数据库）

## 关键规则
1. **全中文命名** — 函数、变量、文件夹、配置全中文
2. **零依赖** — 不pip install任何包，纯标准库
3. **相对路径优先** — 所有路径使用相对路径，复制到任何位置都能跑
4. **文件权限铁律** — 操作新文件夹必须弹窗询问，无权限绝对不动
5. **隐私分区** — 隐私区绝不进入发布包，一键打包只打公共区
6. **大模型全权决策** — 系统不自己设计思路，所有问题丢给大模型
