# 更新日志

## v2.2.1 (2026-06-29)

### 新增
- **文件管理器移动/复制**：后端新增 `移动()` 和 `复制()` 方法 + `/api/file-move` `/api/file-copy` 端点，支持文件/文件夹移动和复制，含权限校验+冲突检测+审计日志
- **缩略图悬浮操作按钮**：画廊缩略图hover时右上角出现🗑️删除/📦移动到/📋复制到三个圆形按钮，点击即可操作
- **HTML5拖拽移动/复制文件**：拖拽文件到左侧文件夹树或中间画廊文件夹=移动；Shift+拖拽=复制；支持单个或框选多个文件批量操作
- **画廊分页渲染**：文件夹内容过多时每页只渲染200项，底部出现「点击加载更多」按钮，彻底解决卡死问题
- **去背景工具**：独立页面（去背景工具.html），上传图片自动去背景生成透明PNG，可从图片查看器一键打开
- **我的电脑分组显示**：快捷访问（桌面/文档/下载/图片）和磁盘（C/D/E...带空间信息）分组展示，左侧树和中间画廊同步

### 修复
- 我的电脑点击后空白：后端有重复的旧版`/api/drives`路由（PowerShell版返回类型字段不匹配），已删除
- 左侧文件树磁盘大小被CSS flex挤掉不可见：改用`.ti-space`(flex-shrink:0)固定空间
- 框选与拖拽冲突：鼠标落在gallery-item上时不启动框选，让HTML5原生拖拽接管

## v2.2.0 (2026-06-27)

### 新增
- **Tavily智能搜索引擎集成**：网络搜索、网页抓取、网页分析三个操作全面接入Tavily API，返回AI优化的结构化结果（标题+URL+正文摘要+相关度评分+AI总结答案），无密钥时自动回退Bing/urllib
- **工具密钥管理界面**：设置→模型→工具密钥卡片，可在界面上查看Tavily密钥状态（掩码显示）、输入保存（加密存储），无需手动编辑JSON
- **后端 /api/tool-keys 端点**：GET返回工具密钥状态，POST保存加密
- **Linux安装支持**：install.sh一键安装脚本（检查Python→clone仓库→创建隐私区目录→初始化密钥→桌面快捷方式）、打包linux.py生成tar.gz便携包
- 流式渲染彩色callout样式（成功/错误/步骤/提示/测试五色，与最终渲染一致）
- 股票预测操作（LSTM神经网络，全JSON配置驱动）

### 优化
- callout样式柔化（降低饱和度+透明度，视觉更舒适）
- 密钥模板同步实际加密格式（多模型嵌套dict结构）

## v2.1.0 (2026-06-26)

### 新增
- 静默自动更新：启动3秒后后台检查，多源回退（Gitee优先→GitHub→自建服务器）
- EXE打包：PyInstaller一键打包，用户无需安装Python
- 更新检查器：GitHub Release + Gitee发行版自动检测，下载覆盖公共区+自动备份+失败回滚
- Gitee镜像仓库：国内用户更新走Gitee，不依赖翻墙

### 修复
- 运行命令超时卡死：subprocess.run超时后子进程成孤儿，改用Popen+taskkill /T杀进程树
- LLM工具调用参数截断：max_tokens不足导致JSON不完整，json.loads失败后静默返回{}→AI收到"缺少必填参数"误报。现在检测finish_reason=length并提示AI拆分文件用追加文件分批写入
- max_tokens 4096→8192，从源头减少截断
- 跨对话记忆混淆：记忆注入标签"相关记忆""近期摘要"改为明确标注"非当前对话历史，不要混入当前对话"
- Ctrl+A被拦截：焦点在对话输入框/编辑器时onSelectionKeyDown仍拦截，增加TEXTAREA/INPUT检查+mediaView可见性检查

### 优化
- 更新源优先级：Gitee优先（国内用户），GitHub次之，自建服务器兜底
